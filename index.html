Google Docs

Google Docs izmantoju visdažādāko dokumentu rakstīšanai - no esejām līdz matemātikas mājasdarbiem līdz 100 lapu gariem dokumentiem. Šajā lietotnē ir arī laba sadarbošanās funkcionalitāte, kas ļoti noder grupu darbu rakstīšanā. Iemesls, kāpēc izvēlos Google Docs nevis Microsoft Word ir ērtība - Google Docs veidotie faili automātiski glabājas manā Google kontā, kamēr Microsoft Word dokumentus man būtu vispirms jāaugšupielādē mākonī.

Google Slides

Klasika! Lai gan studēju Datorzinātnes, tāpatās sanāk diezgan regulāri veidot prezentācijas (šis turpinās arī ārpus universitātes, strādājot par programmētaju). Priekšrocības tādas pašas kā Google Docs - var lietot no jebkuras ierīces, tā kā fails atrodas mākonī, ir viegli sadarboties ar citiem pie viena dokumenta un ir par brīvu. Kā arī liels pluss ir tāds, ka internetā var atrast daudz dažādas prezentāciju sagataves, kuras var izmantot sev nepieciešamām prezentācijām, un tās vislabāk strādā, manuprāt, tieši uz Google Slides.

Google Sheets

Nu ko, pabeigsim mūsu svēto trīsvienību ar pēdējo manis izmantoto Google lietotni. Datorzinātņu studiju programmā pietiekami bieži sanāk strādāt ar datiem. Dažreiz tas ir kursa prasību dēļ, piemēram, statistikas kursā jāpēta tabulu dati. Dažreiz tas ir tādēļ, lai apmierinātu savu ziņkārību, piemēram, veidojot un pētot tabulas par mutiskā eksāmena biļetēm un to, kā tās tiek izlozētas (pilnīgi nejauši, starp citu). Tās pašas priekšrocības, protams, kā Google Docs un Google Slides.

draw.io jeb diagrams.net

Studējot Datorzinātņu programmā, ir brīži, kad tev vajag vizuāli parādīt kādas programmas darbību, datubāzes uzbūvi vai jebko citu. Draw.io ir lielisks bezmaksas rīks šim nolūkam - tas ļauj ērti un diezgan intuitīvi izveidot dažāda veida grafus. Bonuss šai lietotnei ir tāds, ka tā ļauj savus darbus ieglabāt uzreiz savā Google kontā. Šis gan būs vienīgais pluss, kas sakrīt ar iepriekšminētajām lietotnēm, jo diemžēl reāllaika sadarbība ar citiem kolēģiem nav tik laba, cik gribētos.

Visual Studio Code

Ir daudz dažādu teksta redaktoru, katrs savam mērķim vai programmēšanas valodai, bet, manuprāt, Visual Studio Code ir viss, kas tev ir vajadzīgs. Visual Studio Code stiprā puse ir tā adaptivitāte - ja tev kaut kas pietrūkst, tu vari ļoti viegli un ātri ielādēt spraudni / paplašinājumu, kas būs atrisinājis tavu problēmu, piemēram, koda analīze kādai specifiskai programmēšanas valodai. Ja nu tāda paplašinājuma vēl nav, tad tu vari uzrakstīt un publicēt pats savu, jo Visual Studio Code ir atvērtā pirmkoda lietotne.
